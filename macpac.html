Usage

macpac -i[d] <adapter> [[-s <MAC> | random] | -u] [-nofix] [-noreset]
[-add | -del] [-n <name>] [-r | -d | -e] [-show] [-gen] [/?]

-i[d] <adapter>......The network adapter's Network Connections name or ID.
-s <MAC> | random....Set the adapter's MAC address.
-u...................Unset the custom MAC address (if any).
-gen.................Generate a random MAC address.
-nofix...............Don't force locally administered unicast address.
-noreset.............Suppress adapter reset after successful MAC set/unset.
-add.................Add Network Address property to adapter configuration
.........................in Advanced tab.
-del.................Remove Network Address property.
-n <name>............Sets the connection name of the adapter.
-r...................Reset the specified network adapter.
-d...................Disable the specified network adapter.
-e...................Enable the specified network adapter.
-show................Show information on the adapter.
